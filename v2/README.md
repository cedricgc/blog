# My Gatsby Blog (v2)

This is a modern blog site built with [GatsbyJS](https://www.gatsbyjs.com/), a React-based framework for creating fast, optimized websites.

## Development

1. Clone this repository
2. Navigate to the v2 directory: `cd v2`
3. Install dependencies: `npm install`
4. Start the development server: `npm run develop` or `npm start`
5. Open your browser at `http://localhost:8000`

## Available Scripts

- `npm run develop` - Starts the development server
- `npm start` - Alias for `npm run develop`
- `npm run build` - Builds the production-ready site
- `npm run serve` - Serves the production build locally
- `npm run clean` - Cleans the cache and public directories

## Project Structure

- `/content` - Blog posts and other content
- `/src` - Source code
  - `/components` - React components
  - `/pages` - Page templates
  - `/templates` - Templates for programmatically created pages
- `gatsby-config.js` - Main Gatsby configuration file
- `gatsby-node.js` - Gatsby Node API implementations
